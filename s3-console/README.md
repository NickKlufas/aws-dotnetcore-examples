# S3 ConsoleApp README

This project demonstrates how to integrate a DotNet Core Console application with Amazon S3 using the Amazon SDK.

## Features

The following features have been implemented:

* List all S3 buckets
* Create new S3 bucket
* Delete S3 bucket

## Developed With
